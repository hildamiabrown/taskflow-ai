# TaskFlow AI

TaskFlow AI is an AI-powered productivity assistant that automates task planning, prioritization, and daily scheduling. It uses structured prompt-style logic to score tasks by impact, urgency, effort, and energy fit, then generates a clear daily plan.

## Features

- Add and organize tasks with impact, effort, deadline, and energy context
- Simulate API-based decision-making with a transparent task optimization score
- Generate a dynamic daily schedule from prioritized tasks
- Update prompt-style recommendations as task context changes
- Use a responsive interface that works on desktop and mobile

## Project Files

- `index.html` - main application layout
- `styles.css` - responsive visual design
- `script.js` - task scoring, scheduling, and prompt workflow logic
- `README.md` - project overview for GitHub

## How to Run

Open `index.html` in a browser, or use GitHub Pages after uploading the repository.
